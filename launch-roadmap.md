# launch roadmap

## week 1 - backend and auth

- Create Supabase project.
- Run `supabase/schema.sql`.
- Configure Auth email/password.
- Add founder/facilitator/member roles.
- Seed first member/event/payment data.
- Verify RLS policies with test accounts.

## week 2 - founder dashboard

- Connect Streamlit to Supabase.
- Build live KPI metrics.
- Ship member management.
- Ship event builder.
- Add revenue analytics and conversion funnel.
- Add messaging center and risk alerts.

## week 3 - member and facilitator apps

- Deploy Next.js to Vercel.
- Build member event browsing and booking flow.
- Build profile, referrals, and feedback.
- Build facilitator roster/check-in/incident tools.
- Validate mobile UX.

## week 4 - AI, polish, deploy

- Add member-event fit scoring.
- Add churn prediction.
- Add demand forecast.
- Add feedback sentiment summaries.
- Generate personalized invites.
- Final QA, analytics review, launch copy, deployment.
