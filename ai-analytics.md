# ai and analytics layer

## ai features

Member-event fit score:
- Inputs: member tier, comfort level, presence tag, attended sessions, event vibe, event tier.
- Output: 0-100 score and short explanation.

Churn prediction:
- Inputs: last attendance date, booking cancellations, feedback sentiment, message engagement.
- Output: low/medium/high churn risk.

Event demand forecast:
- Inputs: event type, tier, price, venue, weekday, past confirmation rates.
- Output: expected demand and recommended capacity.

Feedback sentiment summary:
- Inputs: post-event feedback and facilitator notes.
- Output: positive themes, friction themes, safety flags, next action.

Auto personalized invites:
- Inputs: member profile, past attendance, event details.
- Output: short warm invite copy in Nook voice.
